# 行为准则

## 我们的承诺

为了建设一个开放、友好的社区，我们承诺尊重所有参与项目的用户和贡献者，无论年龄、身体尺寸、能力、经验水平、教育程度、社会地位、国籍、个人外貌或宗教信仰如何。

## 我们的标准

### 促进积极行为

- 对其他社区成员表现出友好和体贴
- 以专业的方式表达不同意见
- 接受建设性的批评并从中学习
- 关注对社区最有利的事情
- 表达感激之情

### 不可接受的行为

- 使用带有性暗示的语言或图像，以及任何形式的性骚扰
- 在讨论或任何其他社区互动中发表挑衅、侮辱或仇恨言论
- 公开或私下骚扰
- 发布他人的私人信息（如个人地址或电子邮箱），未经明确允许
- 在专业环境被认为不适当的其他行为

### 我们的责任

项目维护者有责任明确什么是可接受的行为，并应根据公认的行为标准采取适当和公平的纠正措施。

项目维护者有权利和义务删除、编辑或拒绝不符合本行为准则的评论、提交、代码、维基编辑、问题和其他贡献，并为任何他们认为不适合的情况提供原因。

## 适用范围

本行为准则适用于所有项目空间和公共场所，当个人代表项目或其社区时。代表项目或社区的例子包括使用官方项目电子邮件地址、通过官方社交媒体账号发布信息，或在任何在线或离线活动中担任指定代表。

## 执行

如果发生虐待、骚扰或其他不可接受的行为，可以联系项目团队。所有投诉都将被审查和调查，并给出必要和适当回应的事件。项目团队有义务对举报人保密。

## 处罚

对于违反本准则的贡献者，项目维护者可采取以下措施：

1. **口头警告** - 解释违规行为
2. **书面警告** - 记录违规详情
3. **临时禁止** - 在指定期限内禁止参与
4. **永久禁止** - 永久禁止参与项目

## 指导原则

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/) 版本 2.1，可在 https://www.contributor-covenant.org/version/2/1/code_of_conduct.html 获取。

指导原则改编自 Mozilla 的行为准则。
